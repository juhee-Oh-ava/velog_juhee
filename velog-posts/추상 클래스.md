<h4 id="추상-클래스">추상 클래스</h4>
<p>new 연산자를 이용해서 객체를 만들지 못하고, 상속을 통해 자식 클래스만 만들 수 있다.</p>
<ul>
<li>추상 클래스<pre><code class="language-java">package sec02;
</code></pre>
</li>
</ul>
<p>public abstract class Phone {</p>
<pre><code>// 필드
public String owner;

// 생성자
public Phone(String owner){
    this.owner = owner;
}

// 메소드
public void turnOn() {
    System.out.println(&quot;폰 전원을 켭니다.&quot;);
}
public void turnOff() {
    System.out.println(&quot;폰 전원을 끕니다.&quot;);
}</code></pre><p>}</p>
<pre><code>
* 실체 클래스
```java
package sec02;

public class SmartPhone extends Phone{

    // 생성자
    public SmartPhone(String owner) {
        super(owner);
    }
    // 메소드
    public void internetSearch() {
        System.out.println(&quot;인터넷 검색을 합니다.&quot;);
    }
}
</code></pre><ul>
<li>실행 클래스<pre><code class="language-java">package sec02;
</code></pre>
</li>
</ul>
<p>public class PhoneMain {
    public static void main(String[] args) {
        // Phone phone = new Phone();
        SmartPhone smartPhone = new SmartPhone(&quot;홍길동&quot;);</p>
<pre><code>    smartPhone.turnOn();
    smartPhone.internetSearch();
    smartPhone.turnOff();
}</code></pre><p>}</p>
<pre><code>
#### 추상 메소드 재정의

* 추상 메소드 선언
```java
package sec03;

public abstract class Animal {
    public String kind;

    public void breathe() {
        System.out.println(&quot;Can you feel my heartbeat?&quot;);
    }

    // 추상 메소드
    public abstract void sound();
}
</code></pre><ul>
<li>추상 메소드 재정의(Dog)<pre><code class="language-java">package sec03;
</code></pre>
</li>
</ul>
<p>public class Dog extends Animal {
    public Dog() {
        this.kind = &quot;포유류&quot;;
    }</p>
<pre><code>@Override
public void sound() {
    System.out.println(&quot;멍멍&quot;);
}</code></pre><p>}</p>
<pre><code>* 추상 메소드 재정의(Cat)
```java
package sec03;

public class Cat extends Animal{
    public Cat() {
        this.kind = &quot;포유류&quot;;
    }

    @Override
    public void sound() {
        System.out.println(&quot;야옹&quot;);
    }
}</code></pre><ul>
<li>실행 클래스<pre><code class="language-java">package sec03;
</code></pre>
</li>
</ul>
<p>public class AnimalMain {
    public static void main(String[] args) {</p>
<pre><code>    // 방법 1 : 자식 타입 변수로 자식 객체 사용
    Dog dog = new Dog();
    Cat cat = new Cat();

    System.out.println(&quot;강아지 종류: &quot; + dog.kind);
    System.out.println(&quot;고양이 종류: &quot; + cat.kind);
    dog.sound();
    cat.sound();
    dog.breathe();
    cat.breathe();


    // 방법 2 : 부모 타입으로 자식 객체 참조
    Animal animal = null;
    animal = new Dog();
    animal.sound();
    animal = new Cat();
    animal.sound();

    // 방법 3: 다형성
    animalSound(new Dog());
    animalSound(new Cat());
}

public static void animalSound(Animal animal){
    animal.sound();
}</code></pre><p>}</p>
<p>```</p>
<blockquote>
<p>추상 클래스는 객체는 못 만들지만, <strong>타입</strong>으로는 사용할 수 있다.
그래서 부모 타입으로 자식 객체를 참조할 수 있다.</p>
</blockquote>
