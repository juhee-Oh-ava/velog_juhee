<h4 id="연산의-방향과-우선순위">연산의 방향과 우선순위</h4>
<pre><code>x &gt; 0 &amp;&amp; y &lt; 0
100 * 2 / 3 % 5</code></pre><blockquote>
<p>대부분의 연산자는 왼쪽에서 오른쪽방향으로 연산을 수행한다.</p>
</blockquote>
<blockquote>
<p>하지만 단항 연산자(++, --, ~, !), 부호 연산자(+, -), 대입 연산자(=, +=, -=) 는 오른쪽에서 왼쪽 방향으로 연산을 수행한다.</p>
</blockquote>
<pre><code>a = b = c = 5;</code></pre><p>이 연산식은 c = 5, b = c, a = b 순서로 실행된다. 실행된 후에는 a, b, c 의 값이 모두 5가 된다.</p>
<h4 id="연산자의-종류">연산자의 종류</h4>
<blockquote>
<p>단항 연산자: ++;
이항 연산자:  x + y;
삼항 연산자 : (sum &gt; 90) ? &quot;A&quot; : &quot;B&quot;;</p>
</blockquote>
