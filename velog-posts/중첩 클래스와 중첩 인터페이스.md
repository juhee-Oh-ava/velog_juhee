<h3 id="중첩-클래스">중첩 클래스</h3>
<p>인스턴스 멤버 클래스 : static 키워드 없이 중첩 선언된 클래스
정적 멤버 클래스 : static 키워드로 선언된 클래스
로컬 클래스 : 메소드 내에서 선언하는 클래스</p>
<pre><code class="language-java">package sec01;

// 바깥 클래스
public class A {
    A() { System.out.println(&quot;A 객체가 생성됨&quot;);}


    // 인스턴스 멤버 클래스
    class B {
        B() { System.out.println(&quot;B 객체가 생성됨&quot;);}
        int field1;
        static final int field2 = 70;
        void method1() {
            System.out.println(&quot;B 의 메소드&quot;);
        }
        // static void method2() { }
    }

    // 정적 멤버 클래스
    static class C {
        C() { System.out.println(&quot;C 객체가 생성됨&quot;);}
        int field1;
        static int field2;
        void method1() {
            System.out.println(&quot;C 의 메소드&quot;);
        }
        static void method2() {
            System.out.println(&quot;C 의 정적 메소드&quot;);
        }
    }

    void method() {
        // 로컬 클래스
        class D {
            D() { System.out.println(&quot;D 객체가 생성됨&quot;);}
            int field1;
            static int field2;
            void method1() {
                System.out.println(&quot;D 의 메소드&quot;);
            }
            static void method2() {
                System.out.println(&quot;D의 정적 메소드&quot;);
            }
        }

        D d = new D();
        d.field1 = 3;
        d.method1();
    }
}
</code></pre>
<pre><code class="language-java">//main
package sec01;

public class Amain {
    public static void main(String[] args) {
        A a = new A();
        A.B b = a.new B(); // 인스턴스 멤버 클래스 객체 생성
        b.field1 = 10;
        System.out.println(b.field1);
        b.method1();

        // static final int field2 = 70;
        System.out.println(A.B.field2);


        A.C c = new A.C();  // 정적 멤버 클래스 객체 생성
        c.field1 = 20;
        A.C.field2 = 30;
        c.method1();
        A.C.method2();


        // 로컬 클래스 객체 생성을 위한 메소드 호출
        a.method();

    }
}

</code></pre>
<blockquote>
<ul>
<li>내부 클래스의 객체에 접근할 때는 외부 클래스의 객체를 만들어 접근한다.</li>
</ul>
</blockquote>
<ul>
<li>인스턴스 멤버 클래스에서는 static 변수와 static 메소드를 사용할 수 없지만, static final 변수는 사용 가능</li>
<li>인스턴스 멤버 클래스 객체 생성 : A.B b = a.new B();</li>
<li>정적 멤버 클래스 객체 생성 : A.C c = new A.C();</li>
<li>로컬 클래스 객체는 method() 가 실행되면 new D()가 실행되고, D 객체 생성가 생성된다. </li>
<li>로컬 클래스의 필드는 main에서 직접 접근할 수 없다. </li>
</ul>

<h3 id="중첩-인터페이스">중첩 인터페이스</h3>
<p>중첩 인터페이스는 클래스의 멤버로 선언된 인터페이스를 말한다.</p>
<pre><code class="language-java">class A {
    [static] interface I {
    void method();}}</code></pre>
<p>중첩 인터페이스는 인스턴스 멤버 인터페이스와 정적 멤버 인터페이스 모두 가능하다. 주로 정적 멤버 인터페이스를 많이 사용하는데 UI 프로그래밍에서 이벤트를 처리할 목적으로 많이 활용된다.</p>
<pre><code class="language-java">package sec01;

public class Button {
    OnclickListener listener;

    void setOnClickListener(OnClickListener listener) {
        this.listener = listener;
    }

    void touch() {
        listener.onClick();
    }

    static interface OnClickListener {
        void onClick();
    }
}
</code></pre>

            @Override
            void order() {
                System.out.println(name + &quot;가 주문되었습니다.&quot;);
